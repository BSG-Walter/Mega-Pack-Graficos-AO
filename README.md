## 🌍 ¿Qué es este repositorio?

Una recopilación masiva de gráficos para servidores de **Argentum Online**: cascos, sombreros, armas, armaduras, NPCs, Objetos, entradas a dungeons y mucho más.

Este proyecto **pertenece a la comunidad** — no tiene un único dueño. Se mantiene vivo gracias a las contribuciones de personas como tú.

## 🚀 ¡CONTRIBUYE! Este repo es de todos

> ### 💡 ¿Tienes gráficos que no están aquí? ¡Envía un Pull Request!
>
> Este repositorio está abierto a **cualquier persona** de la comunidad.  
> No importa si aportas 1 gráfico o 100 — toda contribución es bienvenida.

### ¿Cómo colaborar?

1. Haz un **fork** de este repositorio
2. Añade tus gráficos en la carpeta correspondiente (o crea una nueva si hace falta)
3. Haz un **commit** con una descripción clara de lo que añades
4. Abre un **Pull Request** hacia `main`
5. ¡Listo! Lo revisaremos y lo mergeamos

### Pautas básicas

- Los archivos deben ser **compatibles con Argentum Online**
- Nombra los archivos de forma descriptiva (evita `grafico1.bmp`, prefiere `casco_vikingo.bmp`)
- Si el gráfico tiene un autor conocido, indícalo en la PR o en un `CREDITOS.txt` dentro de la carpeta
